Brief description of submitted projects:
FoothillCSC/randomArray
 - mini project that generates an array of random integers. Asked to minimize the colisions of values in the array.
 Generate random number between 0 and twice the size of array (following the hash collision ideal)

FoothillCSC/week02
 - finds optimal combination of items (song + groceries) to fit within a given budget (duration for songs, price for
 groceries). Uses a subset sum algorithm to find the best combination of items that is closest to the given budget
 without going over.

FoothillCSC/week03
 - uses stacks to simulate browser navigation. Uses a stack of links to keep track of the current page and the
  previous pages. Allows user to navigate back and forward through the stack of links.

FoothillCSC/week04
    - implements a lazy search tree to manage inventory items. Allows for soft deletion of items.

FoothillCSC/week05
    - similar to week04 but with added functionality for hard deletion of items from the lazy search tree.
Also includes error handling for invalid removal entries.

FoothillCSC/week06
    - implements a search functionality for destinations using a hash map to store destination data and a linked list
to manage search results. Allows users to search for specific words in the destination descriptions and retrieve
relevant. Implements a GUI to display search results in a user-friendly interface.

FootillCSC/week07
    - find optimal recursion limit for quick sort algorithm. Tests different recursion limits on large array sizes to
find the optimal one for performance.
